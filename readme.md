### 这是一个基于react的ui库 --Free-ui
### 参考了各大ui插件，如element-ui，antd等等，有自己的一些创新。

使用 
```
npm install --save lyl-aaa-ui

or

yarn add lyl-aaa-ui
```

### 支持typescript

### 使用
import {Button} from 'free-ui'

### jsx的形式使用
```
<Button></Button>
```
